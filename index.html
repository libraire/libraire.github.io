Gitbook的使用

需详细了解请戳指南, 常用操作如下:


安装gitbook
# npm install gitbook-cli -g

在SUMMARY.md中编辑目录,并通过下面命令生成对应的文件和文件夹
# gitbook init

生成静态文件
# gitbook build

启动本地服务
# gitbook serve
